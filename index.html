📤 ¿Cómo publicar un nuevo audio?

  1. Coloca el archivo .m4a en la carpeta audios/ de tu repositorio local.
  2. Agrega el episodio desde este panel (los datos se guardan en localStorage).
  3. Ejecuta en terminal: 
    git add audios/mi_audio.m4a
git commit -m "Agregar nuevo episodio: Título"
git push
  4. La app cargará el audio desde la red; el Service Worker lo cacheará para offline.

⚠️ Recuerda que el archivo debe ser accesible en la misma ruta relativa (audios/) en producción.
